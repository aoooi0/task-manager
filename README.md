# 📝 タスク管理アプリ

効率的にタスクを管理するシンプルで使いやすいWebアプリケーションです。

## 🎯 機能

### コア機能
- ✅ **タスク追加** - 新しいタスクを簡単に追加
- ✏️ **タスク編集** - 既存のタスクをいつでも編集可能
- 🗑️ **タスク削除** - 不要なタスクを削除
- ✔️ **タスク完了** - チェックボックスで完了状態を管理

### 優先度管理
- 🔴 **高** - 緊急度の高いタスク
- 🟡 **中** - 通常のタスク
- 🟢 **低** - 優先度の低いタスク

### 期限管理
- 📅 **期限日設定** - タスクに期限日を設定可能
- ⚠️ **期限超過警告** - 期限を超過したタスクは赤色で表示

### フィルター機能
- 📋 **すべて表示** - 全タスクを表示
- ⏳ **未完了のみ** - 完了していないタスクのみ表示
- ✨ **完了済みのみ** - 完了したタスクのみ表示

### ソート機能
- 📌 **追加日時順** - 最新に追加したタスクから順に表示
- 📅 **期限日順** - 期限が近いタスクから順に表示
- ⭐ **優先度順** - 高い優先度のタスクから順に表示

### 統計情報
- 📊 **総タスク数** - 登録されているすべてのタスク数
- ⏳ **未完了タスク数** - 完了していないタスク数
- ✅ **完了済みタスク数** - 完了したタスク数

### クイックアクション
- 🧹 **完了済みを削除** - 完了したタスクをまとめて削除
- 🔄 **すべて削除** - すべてのタスクをクリア

## 🚀 使い方

### 1. タスクを追加
1. 上部の入力欄にタスク名を入力
2. 優先度をドロップダウンから選択（低・中・高）
3. 期限日をカレンダーから選択（オプション）
4. 「追加」ボタンをクリック
5. またはEnterキーを押す

### 2. タスクを完了
- タスクの左側のチェックボックスをクリック
- 完了したタスクは薄く表示されます

### 3. タスクを編集
- タスクの右側の ✏️ ボタンをクリック
- タスク名、優先度、期限日を編集
- 「保存」をクリック

### 4. タスクを削除
- タスクの右側の 🗑️ ボタンをクリック
- 確認ダイアログで「OK」をクリック

### 5. フィルター機能を使用
- 「すべて」「未完了」「完了済み」ボタンで表示を切り替え

### 6. ソート機能を使用
- 「並び替え」ドロップダウンから並べ替え方法を選択

## 💾 データ保存

- すべてのタスクはブラウザの **LocalStorage** に自動保存されます
- ブラウザを閉じてもタスクデータは保存されています
- キャッシュをクリアするとデータも削除されます
